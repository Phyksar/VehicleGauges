# VehicleGauges
A vehicle gauges for Grand Theft Auto 4 (1.2.0.43) .NET ScriptHook (0.5.1)

## Installation
- Make sure the ScriptHook is installed and working
- Place .cs and .ini files in the `scripts` folder of your game directory
